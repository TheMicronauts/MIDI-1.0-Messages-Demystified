# MIDI 1.0 Messages Cheat Sheets
The resources in this repo untangle the essentials of the MIDI 1.0 standard for the beginner musician or aspiring instrument maker.

The mental model is that of a musician and producer. I invite developers to approach MIDI with this mental model if they wish to stop writing wonky programs that are hostile to musical practice.

[MIDI 1.0 Messages Cheat Sheet](https://docs.google.com/document/d/1mNEl-4xRUtdilLxmV_6JKfgVAksGtDIJLRqTvIcyQwY/): lists the different MIDI messages and their names, along with a brief explanation

[MIDI 1.0 CC Cheat Sheet](https://docs.google.com/spreadsheets/d/1dl6iuqFCuGr4E6kiuZF266JmtjZGssYR-wN0yGWjE9w/): Lists the 128 Control Change messages, along with their names and useful information
